Gestor Tipo Características
MySQL Relacional (SQL) Multihilo y multiusuario utilizado en un gran número de páginas web,software libre ,multiplataforma y soprte SSL.
MariaDB Relacional (SQL) " " ", compatibilidad MySQL ,gran escalabilidad y rendimiento.

Oracle Database Relacional (SQL) " " ",privativo,robustez, Eficiencia y Estabilidad.
SQLite Relacional (SQL) Pequeñas BD,uso aplicaciones mobiles,Agendas whatapps etc.
NoSQL Las bases de datos NoSQL se crean para modelos de datos específicos y tienen esquemas flexibles que permiten a los programadores crear y administrar aplicaciones. NoSQL también es más ágil porque no se basa en el concepto de tabla y no usa SQL para manipular o analizar datos (aunque algunas bases de datos NoSQL pueden tener un lenguaje de consulta inspirado en SQL).
    Flexibilidad: permiten un desarrollo más rápido y más iterativo. Escalabilidad: diseñadas para escalar usando clústeres distribuidos
    de hardware en lugar de escalar añadiendo servidores. Alto rendimiento: optimizada para un mayor rendimiento y lograr una funcionalidad similar con bases de datos relacionales.
MongoDB No relacionales (NoSQL) Codigó abierto,estructuras BSON,dinámico ,facilidad de integración.
Cassandra No relacionales (NoSQL) Compresión de datos, sin que ello suponga un gasto de recursos. CQL (Lenguaje de Consulta Cassandra): un lenguaje similar a SQL.
