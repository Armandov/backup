¿QUÉ ES UNA DISTRIBUCIÓN LINUX?

Una distribución de Linux, a menudo abreviada como "distro de Linux", es una versión del sistema operativo Linux de código abierto que se incluye con otros componentes, como programas de instalación, herramientas de administración y software adicional.

¿POR QUÉ TODAS ESAS DISTRIBUCIONES SON LINUX , SI SON SISTEMAS OPERATIVOS DIFERNTES?

Debido a que Linux se puede redistribuir libremente.Varias organizaciones empaquetan Linux, a menudo combinándolo con aplicaciones gratuitas o propietarias. Dicho paquete que incluye todo el software que necesita para instalar y ejecutar Linux.

¿CUALES SON LAS PRINCIPALES RAMAS DE DICHAS DISTRIBUCIONES?

  • Sistema operativo de bajo costo o libre.
  • Una plataforma informática estable y confiable.
  • Una plataforma informática de alto rendimiento.
  • Para una gran red ,organizaciones gobiernos o empresas.

¿CUALES SON LAS VENTAJAS DE USAR LINUX?

  • Linux es un sistema operativo multiplataforma que se ejecuta en muchos modelos de computadora. Solo Unix, un antepasado de Linux, rivaliza con Linux a este respecto.
  • Linux es libre, en todos los sentidos.Es posible obtener y usar Linux libremente.Se puede optar por comprar Linux a un proveedor que incluya Linux con documentación o aplicaciones especiales, o que brinde soporte técnico.
  • Linux y muchas aplicaciones de Linux se distribuyen en codigó fuente.Esto hace posible que usted y otros puedan modificarlos o mejorarlos.
  • Linux tiene características y rendimiento atractivos.El acceso libre al código fuente de Linux permite a los programadores de todo el mundo implementar nuevas funciones y ajustar Linux para mejorar su rendimiento y confiabilidad.
