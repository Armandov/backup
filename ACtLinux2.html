LINUX

En Linux y Unix todo es un fichero.Los directorios son ficheros, los ficheros son ficheros, y los dispositivos son ficheros.
A veces a los dispositivos se les llama nodos, pero siguen siendo ficheros.
Los sistemas de ficheros de Linux y Unix se organizan en una estructura jerárquica, de tipo árbol.
El nivel más alto del sistema de ficheros es / o directorio raíz.
Todos los demás ficheros y directorios están bajo el directorio raíz.
/The Root Directory » Todo el sistema Linux se encuentra en el directorio /, conocido como el directorio raíz.
/bin » contiene los binarios (programas) esenciales del usuario que deben estar presentes cuando el sistema se monta en modo de usuario único.Aplicaciones
/boot » contiene los archivos necesarios para arrancar el sistema, Scripts o directorios de scripts ; por ejemplo, los archivos del cargador de arranque y los núcleos de Linux.
/dev » contiene archivos especiales que representan dispositivos. Estos no son archivos , se muestran como archivos; ejemplo: /dev /sda.
/etc » contiene archivos de configuración, generalmente se pueden editar.
/home » contiene una carpeta de inicio para cada usuario y los archivos de datos del mismo.
/lib » contiene librerías compartidas necesarias para los programas que se encuentran en el sistema de archivos raíz./bin y /sbin.
/media » contiene subdirectorios donde se montan los dispositivos de medios extraíbles insertados.
/mnt » contiene puntos de montaje temporales,ra los sistemas de archivos montados por el administrador del sistema.
/opt » contiene subdirectorios para paquetes de software opcionales , donde instalar aplicaciones opcionales (de terceros).
/proc » directorio dinámico especial que mantiene información sobre el estado del sistema, incluyendo los procesos actualmente en ejecución
/root » directorio personal del usuario root (superusuario);
/run »contiene archivos transitorios para procesos.
/sbin »contiene binarios de administración del sistema ,destinados a ser ejecutados por el usuario root para la administración del sistema.
/srv »contiene datos para los servicios proporcionados por el sistema. HTTP ,NTTP , entre otros.
/sys »contiene contiene archivos que proveen información del kernel a eventos del sistema operativo.
/tmp » contiene archivos temporales Las aplicaciones almacenan estos.Generalmente se eliminan cada reinicia el sistema.
/usr » contiene aplicaciones y archivos a los que puede acceder la mayoría de los usuarios
/var »contiene archivos variables como archivos de registros y bases de datos.
